Kubernetes CronJob

范例

cronjob.yaml

创建 CronJob

kubectl create -f cronjob.yaml

查看 CronJob

kubectl get cronjob hello

关注 Job 状态

kubectl get jobs --watch

删除 CronJob

kubectl delete cronjob hello
